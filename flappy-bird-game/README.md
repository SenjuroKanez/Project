# Flappy Bird Game

This is a simple Flappy Bird clone implemented using Python's Turtle graphics library. The game features a bird that the player controls to navigate through pipes while avoiding collisions.

## Project Structure

```
flappy-bird-game
├── src
│   ├── main.py        # Entry point of the game
│   ├── bird.py        # Bird character logic
│   ├── pipe.py        # Pipe obstacle logic
│   ├── game.py        # Game management and state
│   └── utils.py       # Utility functions
├── requirements.txt    # Project dependencies
└── README.md           # Project documentation
```

## Requirements

To run this project, you need to have Python installed along with the Turtle graphics library. You can install the required dependencies by running:

```
pip install -r requirements.txt
```

## How to Run the Game

1. Clone the repository or download the project files.
2. Navigate to the project directory.
3. Run the main game file:

```
python src/main.py
```

## Features

- Control the bird using the spacebar to jump.
- Avoid the pipes to keep the game going.
- Score points for each pipe successfully passed.

Enjoy playing the game!
